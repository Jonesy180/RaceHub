RaceHub manufacturer wordmark assets

All Garage manufacturer cards use a consistent white wordmark watermark.
Accent colours are applied separately in js/garage.js, keeping the logo artwork monochrome and cohesive.

File names use a lowercase slug of the manufacturer name, for example:
  Aston Martin -> aston-martin.svg
  Mercedes-AMG -> mercedes-amg.svg

SVGs should be white/monochrome with a transparent background. Missing files are
handled automatically: the Garage card simply displays without a watermark.
